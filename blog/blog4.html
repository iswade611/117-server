November 14, 2024

I learned a lot about different CSS layout techniques while reading through these chapters. From image borders, text wrapping around images, and images in the background of text, there are many different ways that you can use CSS and images to spice up the layout of your website.

The one that I chose to focus on for today was the floating technique. A lot of the techniques seemed pretty complex, and I was having trouble executing some of them, so I decided to practice floating since we had already touched upon it in class. Floating the image to the right of the website was pretty self explanatory, and the book helped me figure out that you need to specify the width of the image in the styles page when you’re floating an image. This is something that I wasn’t aware of and will now keep note of for future reference when I use the float technique on my images.
